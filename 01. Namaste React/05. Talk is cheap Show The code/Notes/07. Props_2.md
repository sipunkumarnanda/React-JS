
## üß© Goal: Pass an Object as Props Using `...obj`

In React, you can pass props in multiple ways:

### ‚úÖ Normal way:

```jsx
<Component name="Alice" age={30} />
```

### ‚úÖ Spread way:

```jsx
const user = { name: 'Alice', age: 30 };
<Component {...user} />
```

This spreads each property of the `user` object into individual props.

---

## üîç How the Spread Operator Works

```js
const user = {
  name: "Alice",
  email: "alice@example.com",
  age: 25
};

// becomes:
<Component name="Alice" email="alice@example.com" age={25} />
```

---

## üõ†Ô∏è Full Example: Spread Object as Props

### 1. **Child Component**

```jsx
function UserCard({ name, email, age }) {
  return (
    <div style={{ border: "1px solid #ccc", padding: "1em" }}>
      <h2>{name}</h2>
      <p>Email: {email}</p>
      <p>Age: {age}</p>
    </div>
  );
}
```

### 2. **Parent Component**

```jsx
function App() {
  const user1 = {
    name: "Alice",
    email: "alice@example.com",
    age: 25
  };

  const user2 = {
    name: "Bob",
    email: "bob@example.com",
    age: 30
  };

  return (
    <div>
      <UserCard {...user1} />
      <UserCard {...user2} />
    </div>
  );
}
```

### ‚úÖ Output:

```html
Alice
Email: alice@example.com
Age: 25

Bob
Email: bob@example.com
Age: 30
```

---

## üîÑ Combine Spread Props with Additional Props

You can mix spread props with custom overrides:

```jsx
<UserCard {...user1} age={35} />  // Overrides age to 35
```

> Order matters: later props override earlier ones.

---

## üß† Why Use `...obj` to Pass Props?

### ‚úÖ Pros:

* **Cleaner code**: Especially when you have many props
* **Dynamic UIs**: Great for mapping over arrays of data
* **Reusability**: Easily reuse config/data objects

### ‚ö†Ô∏è Cons:

* **Less explicit**: You may lose track of what‚Äôs being passed
* **May pass unintended props**: If the object has extra keys

## üí° Advanced: Mapping List with Spread Props

```jsx
const users = [
  { name: "Alice", email: "alice@example.com", age: 25 },
  { name: "Bob", email: "bob@example.com", age: 30 }
];

function App() {
  return (
    <div>
      {users.map((user, index) => (
        <UserCard key={index} {...user} />
      ))}
    </div>
  );
}
```
